# How To Run:
1. Download Expo from app store
2. Run ```npm install -g expo-cli```
3. Clone the repo
4. Run ```npm install``` in the repo directory
5. Run ```npm start``` in the repo directory
6. Connect with any QR scanner and open project in Expo app
